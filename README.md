# 🎮 Missão Front-End

**Missão Front-End** é um jogo educativo que ensina conceitos básicos de desenvolvimento web de forma interativa. O jogador deve coletar os elementos corretos de cada tecnologia (HTML, CSS e JavaScript) enquanto evita os incorretos.

---

## 🚀 Tecnologias Utilizadas

* **HTML** → estrutura do jogo
* **CSS** → aparência e layout
* **JavaScript** → lógica e interatividade

---

## 🧠 Lógica do Jogo

O jogo funciona com um sistema de **objetos que caem** do topo da tela, representando elementos das tecnologias estudadas.

O jogador movimenta o personagem para capturar:

* ✔ elementos corretos → ganha pontos
* ❌ elementos errados → perde vidas

O sistema utiliza:

* geração dinâmica de elementos
* detecção de colisão
* controle de estado (`gameAtivo`, `emTransicao`)
* progressão por fases
* transições visuais (*fade*)
* condições de vitória e derrota

---

## 🕹️ Regras

* Use as setas (← →) para se mover
* Você começa com **3 vidas**
* A cada erro, perde uma vida
* O jogo termina ao zerar vidas ou completar todas as fases

---

## ⏫ Progressão

* Fase 1 (HTML) → 10 pontos
* Fase 2 (CSS) → 20 pontos
* Fase 3 (JavaScript) → 30 pontos

🏆 Vitória ao atingir **30 pontos**

---

## 🎉 Tela Final

Ao completar todas as fases, o jogador visualiza uma tela especial que representa um **site construído ao longo do jogo**.

Durante cada fase, o jogador coleta elementos essenciais de **HTML, CSS e JavaScript**. Esses elementos simbolizam a construção de um site completo, desde a estrutura até o estilo e a interatividade.

👉 Ou seja, ao final da jornada, o jogador não apenas vence o jogo, mas também **“constrói” simbolicamente seu próprio site**, aplicando os conceitos aprendidos de forma prática e visual.

---

## 💻 Conceitos Utilizados

### 🟠 HTML — Estrutura

Define os elementos do jogo na tela (divs, áreas, HUD, telas).

### 🔵 CSS — Estilo

Responsável pelo visual, cores, layout e transições.

### 🟡 JavaScript — Lógica

Controla movimentação, colisões, pontuação, fases e estados do jogo.
